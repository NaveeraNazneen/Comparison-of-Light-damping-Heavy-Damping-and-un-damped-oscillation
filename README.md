This code/ plot shows how damping can occur in SHM(Simple Harmonic Motion). In an ideal case there is no friction so the object continues oscillating in between the two extreme positions without any loss in it s amplitude. In case of light damping which in most practical systems happens, the amplitude of oscillations slowly decreases until the object is at rest, as seen in pendulums after being disturbed. In case of critical damping, the object stops in shortest possible time without any oscillations, as the system comes at equilibrium vs in heavy damping the object also stops (as seen in systems with high resistance) but more slowly in comparison to critical damping, with no oscillation. (critical damping not shown)
